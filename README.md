# Social Listening Report

## Overview

This project analyzes online conversations about a brand to understand public sentiment and key discussion topics using social listening techniques.

## Objectives

* Monitor brand mentions across online platforms
* Analyze sentiment (Positive / Neutral / Negative)
* Identify key topics and keywords from discussions

## Tools

* Excel (Data cleaning & analysis)
* PowerPoint (Insight presentation)
* Tableau (Dashboard visualization)
* Looker Studio (Interactive dashboard)

## Key Insights

* Most conversations were **neutral to positive**.
* Positive mentions focused on **product quality and promotions**.
* Negative feedback was mainly related to **service experience**.

## Dashboard

The dashboard visualizes sentiment distribution, mention volume, and key discussion topics.
